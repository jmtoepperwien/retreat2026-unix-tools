# Modern UNIX Terminal Tools & Workflows -- Skill Session

This repository contains introductions to different topics that you can do during this skill session (folders).

We tried to keep each topic short, so that you may combine multiple ones. *Feel free to skip, mix or go completely offboard.* These are just meant as guidelines showing what is possible.

Here is a short introduction to what the topics are about:
